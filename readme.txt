KRSP Frontend File Uplader makes it easy to add a simple and intuitive image and file uploader.

== Description ==

KRSP Frontend File Uploader lets your blog or site users upload images and files. You can manage the files as admin or add them to their profiles etc. However the implementation is all in your hands. The File uploader simply allows any logged in users to upload images that are saved in the media manager as attachments.

Once you've installed the plugin, simply add the shortcode to any page or post where you want your logged in users to be able to add images.

To report bugs or feature requests, [please use included contact page in settings]

== Installation ==

KRSP Frontend File Uploader is installed like any other Wordpress Plugin. Go to plugins menu, and click add new, or manually upload and activate the plugin.

== Frequently Asked Questions ==

= Is there a demo I can see? =

Yes, [Here's our demo](https://krsp.co/krsp-frontend-file-upload-plugin/ "Upload Demor")

= How do manage the plugin settings? =

We've created a simple settings page under Wordpress Settings where you can manage th eplugin upload specific settings. You can change the allowed file formats, as well as the allowed max upload files which defaults to 1.

= How do I upgrade the plugin? =

As new versions become availble, you'll be notified of them in your plugins page, as well as under the dashboard. Simply click the update button/link.
